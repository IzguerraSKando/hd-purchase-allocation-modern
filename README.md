# HD Purchase Allocation — Modern Edition

Beautiful modern recreation of the original HD purchase allocation tool using **shadcn/ui**, Next.js 15, React 19, and Tailwind.

## Super Simple Setup (for non-developers)

1. Click the green **Code** button on this page → **Download ZIP**
2. Unzip the folder on your computer
3. Open the folder in Terminal (or just drag the folder into Terminal)
4. Copy and paste these commands one by one:

```bash
cp .env.example .env
npm install
npx prisma generate
npx prisma db push
npm run dev
```

5. Open your browser and go to: **http://localhost:3000**

That’s it! The modern app will be running.
