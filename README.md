# HD Purchase Allocation 2.0 — Modern Edition

Recreated from the original with cutting-edge shadcn/ui design.

## Stack
- Next.js 15 + React 19
- shadcn/ui + Radix + Tailwind
- next-themes (dark mode)
- Prisma + SQLite
- HD orange branding

## Quick Start
```bash
git clone https://github.com/IzguerraSKando/hd-purchase-allocation-modern.git
cd hd-purchase-allocation-modern
cp .env.example .env
npm install
npx prisma generate
npx prisma db push
npm run dev
```
Open http://localhost:3000

See full README in the project for design notes and remaining files to copy from original for 100% feature parity.
